# Generative AI Flask App Template

This repository provides a minimal Flask-based Generative AI backend.

## Features
- Clean architecture
- Prompt templates as code
- Mock LLM (no API keys required)
- Easy upgrade to local or cloud models
- Perfect for workshops and teaching

## Run the App

```bash
# For windows systems
python -m venv venv
source venv/Scripts/activate
source venv/Scripts/activate
# For Mac or Linux
python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt
python run.py
